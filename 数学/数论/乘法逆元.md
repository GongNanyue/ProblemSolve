逆元存在定理









模板[P3811 【模板】乘法逆元](https://www.luogu.com.cn/problem/P3811)

```cpp
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (a % b == 0) {
        x = 0, y = 1;
        return b;
    } else {
        ll d = exgcd(b, a % b, y, x);
        y -= a / b * x;
        return d;
    }
}

ll inv(ll a, ll m) { //a * x == 1 (mod m) 乘法逆元
    ll x, y;
    exgcd(a, m, x, y);
    x = (x % m + m) % m;
    return x;
}
```
